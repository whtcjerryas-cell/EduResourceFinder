## 💻 使用示例

### 示例1: K12教育搜索 - 印尼一年级数学

**步骤**:
1. 打开 http://localhost:5001/app
2. 确保选中"📚 K12教育"标签
3. 选择国家: **印度尼西亚**
4. 选择年级: **Kelas 1**（一年级）
5. 选择学科: **Matematika**（数学）
6. 输入关键词: **Pecahan**（分数）
7. 点击"🔍 搜索K12资源"

**预期结果**:
- 系统会显示适合一年级学生的分数学习资源
- 搜索查询会自动组合为 "Matematika Pecahan Kelas 1"

### 示例2: 大学教育搜索 - UI计算机科学算法

**步骤**:
1. 点击"🎓 大学教育"标签
2. 选择大学: **UI (印度尼西亚大学)**
3. 选择学院: **FIK (计算机科学学院)**
4. 选择专业: **TI-SKRI (计算机科学)**
5. 选择课程: **CS101 (算法与编程)**
6. 输入关键词: **Algoritma**（算法）
7. 点击"🔍 搜索大学资源"

**预期结果**:
- 系统会在UI的计算机科学专业课程范围内搜索算法相关资源
- 上下文信息会显示完整的层级路径

### 示例3: 职业教育搜索 - Python数据科学

**步骤**:
1. 点击"💼 职业教育"标签
2. 选择技能领域: **💻 IT (信息技术)**
3. 选择培训课程: **IT-DATA (数据科学)**
4. 选择目标受众: **advanced (高级学习者)**
5. 输入关键词: **Python**
6. 点击"🔍 搜索职业教育资源"

**预期结果**:
- 系统会搜索高级Python数据科学培训课程
- 显示提供商(RevoU)、时长(9个月)、认证等信息

## ✨ 核心功能亮点

### 📚 K12教育

- ✅ **智能联动**: 选择年级后自动过滤可用学科
- ✅ **科学配对**: 1-2年级不显示物理化学

## 🔧 API端点

所有功能都可以通过API调用：

```bash
# K12搜索
curl -X POST http://localhost:5001/api/search \
  -H "Content-Type: application/json" \
  -d '{"country":"ID","grade":"Kelas 1","subject":"Matematika","query":"Pecahan"}'

# 大学搜索
curl -X POST http://localhost:5001/api/search_university \
  -H "Content-Type: application/json" \
  -d '{"country":"ID","university_code":"UI","query":"Algoritma"}'

# 职业搜索
curl -X POST http://localhost:5001/api/search_vocational \
  -H "Content-Type: application/json" \
  -d '{"country":"ID","skill_area":"IT","query":"Python"}'
```
