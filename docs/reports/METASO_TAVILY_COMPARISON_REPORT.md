## 9. 优缺点总结

### Metaso

#### ✅ 优点
1. **响应速度快**：平均 0.39s，比 Tavily 快 4.4倍
2. **中文内容优秀**：相关性 0.92，比 Tavily 高 31%
3. **成本优势**：5,000 次免费，之后 ¥0.03/次
4. **API 稳定**：MCP JSON-RPC 2.0 协议，标准化
5. **多种搜索模式**：simple/deep/research，支持学术搜索

#### ❌ 缺点
1. **国际内容质量低**：质量得分仅 0.37
2. **域名匹配弱**：国际教育平台匹配率低（0.2/5）
3. **结果质量不稳定**：有时返回低质量或不相关结果
4. **非英语查询理解差**：印尼语、俄语查询表现不佳

---

### Tavily

#### ✅ 优点
1. **结果质量高**：质量得分 0.72，比 Metaso 高 35%
2. **国际内容优秀**：综合得分高 58%
3. **教育平台匹配强**：擅长找到 YouTube、Khan Academy 等
4. **多语言支持好**：印尼语、俄语查询表现好
5. **域名过滤支持**：原生支持 include_domains 参数

#### ❌ 缺点
1. **响应速度慢**：平均 2.11s，比 Metaso 慢 4.4倍
2. **成本高**：无免费额度，单价 >¥0.03/次

```python
def select_search_engine(query, country_code):
    """
    根据查询和国家选择搜索引擎

    Args:
        query: 搜索查询
        country_code: 国家代码（CN, ID, US, IN, RU等）

    Returns:
        搜索引擎名称（"metaso" 或 "tavily"）
    """
    # 1. 检测查询语言
    is_chinese = detect_chinese(query)

    # 2. 中国/中文查询：优先 Metaso
    if country_code == "CN" or is_chinese:
        return "metaso"  # 相关性高，速度快

    # 3. 国际查询：优先 Tavily
    else:
        return "tavily"  # 质量高，教育平台匹配好
```

```python
    """
```
